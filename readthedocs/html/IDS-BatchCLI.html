IDS-Batch CLI

1. Add IDS-Batch to "PATH" environment variable


Please follow the following instructions to add IDS-Batch CLI to the path environment variable on Windows/Linux.


On Windows


  • Open Advanced system settings from system properties. 





  • Select Environment variables under the advanced tab in system properties. 





  • If PATH variable is already present in the variable list (User/System Variables), then select it and click Edit. 




  • Use semicolon(;) after the existing path and add the path of directory containing IDS-Batch (Eg. “D:\Agnisys\Agnisys Projects\IDSbatch”), then click OK. 


  • If PATH variable is not present in variable list, then create one by clicking on ‘New’. Set variable name as path and variable value as complete path to directory containing IDS-Batch. 


On Linux


  • To add IDS-batch in path variable on linux add the following lines: 


  1. In .bashrc for bash shell 


      %export PATH=<complete path of directory containing IDS-Batch>:$PATH 



  1. In .cshrc for cshell 


     %setenv PATH <complete path of directory containing IDS-Batch>:$PATH 


2.  Error while running Window Installer (on Windows)




The above issue can be resolved by using one of the following two alternatives: 


Option 1 :

    • Run .exe of windows installer as administrator. 


Option 2  : 

    • Follow the following steps 


      1. Click on Start > Run > regedit & press Enter 


      1. Find this key name 


 HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\services\LanmanServer\Parameters 


      1. Locate IRPStackSize


      1. If this value does not exist Right Click on Parameters key and Click on New > Dword Value and type in IRPStackSize under the name. The name of the value must be exactly (combination of uppercase and lowercase letters) the same as what is mentioned above. 


      1. Right Click on the IRPStackSize and click on Modify. 


      1. Select Decimal enter a value higher than 15(Maximum Value is 50 decimal) and Click OK. 


      1. Close the registry editor and restart the computer. 


3.  Word not found


  • On Windows machine, indicates that you do not have Microsoft Word installed. Word is needed if Word files are generated. 

4.  IDS-Batch not found


  • Make sure the IDS-Batch installation directory is in your PATH. On Linux/Solaris machines, check the execute permission of IDS-Batch.

5.  Word file not generated


  • Make sure MS Word in installed on the machine.
