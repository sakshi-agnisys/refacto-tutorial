AMBA-AXI4FULL



IDS supports AMBA-AXI4FULL bus in VERILOG. All the transactions to the registers passes through the AXI4FULL widget. Burst mode is supported. AMBA-AXI4FULL bus is supported at chip and block level.


Axi4full additional signals:


Signal

Width

Driver

Function

Awlen (write address channel signal) / Arlen (read address channel signal)

8

initiator

Burst length. The burst length gives the exact number of transfers in a burst. This

information determines the number of data transfers associated with the address.

Awsize (write address channel signal) / Arsize (read address channel signal)

3

initiator

Burst size. This signal indicates the size of each transfer in the burst.

Awburst (write address channel signal) / Arburst (read address channel signal)

2

initiator

Burst type.

2’b00 : FIXED

2’b01 : INCR

2’b10 : WRAP

2’b11 : RESERVED

Wlast

1

initiator

Write last. This signal indicates the last transfer in a write burst.

Rlast

1

target

Read last. This signal indicates the last transfer in a read burst.

Awid

5

initiator

Write address ID. This signal is the identification tag for the write address group

of signals.

Arid 

5

initiator

Read address ID. This signal is the identification tag for the read address group of signals.
