Assign property



In IDS with help of ‘assign’ property user can assign any field’s or another signal value to the particular signal. This property can be applied at signal and the direction of signal must be output type.


IDSWord



IDSExcel



SystemRDL


property assign { type = string ; component = signal;};

property direction { type = string ; component = signal;};


signal {direction = "out"; assign = "Reg1.F1" ; } S1;


addrmap Block1 {


  reg Reg1 {


    field {

 

       sw = rw;

       hw = rw;


    }F1[31:0] = 32'h0;


  };


 Reg1 Reg1;


};
