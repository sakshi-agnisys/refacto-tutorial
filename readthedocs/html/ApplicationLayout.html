Application Layout

Main Menu:



File, Edit, Help, and Options constitutes the main menu bar which further comprises of many other options.


Toolbar:


Here the IDSNG Tool bar is described as follows:


1. Configure: Upon clicking on configure we get the configuration window.


2. OutDir: OutDir is the output directory where the corresponding outputs gets saved.


3. Reg diff: Upon clicking on this option we can compare any two testcases.


4. Check:  It performs automatic checks. Note that if there are errors, appropriate messages will be displayed next to the source of errors.


5. Generate: It creates all the desired outputs. User can also preview all the generated files in the Project Explorer. This window displays a list of all files and projects that user have created.
