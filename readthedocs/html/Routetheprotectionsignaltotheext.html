Route the protection signal to the external interface


Property name: ext_signal = prot

                                                  

Requirement: This property is used to route the protection checks written in their external CSRs to the external interface through the widget's protection signal.


Applied on : top element(chip/block).


Note :- This property is supported with  AXI, Ambaahb3lite, AMBA, APB  buses. 


Bus Name

Output width

Assigned as

AMBA-AHB

Output [3:0] ext_prot

assign ext_prot = hprot_i

AMBA3AHBLITE

Output [3:0] ext_prot

assign ext_prot = hprot_i

APB

Output [2:0] ext_prot

assign ext_prot = pprot_i

AXI

Output [2:0] ext_rd_prot

Output [2:0] ext_wr_prot

assign ext_rd_prot = arprot_i

assign ext_wr_prot = awprot_i

interface external_interface #(

    parameter bus_width   = 32,

    parameter addr_width  = 2

    );

    

    logic [addr_width-1:0] address_out;

    logic [bus_width-1:0] wr_data_out;

    logic [2:0] ext_prot;

    modport mp (

    output address_out , wr_data_out , ext_prot);

    modport op (

    input address_out , wr_data_out , ext_prot);

endinterface // EXTERNAL
