Hide property English Description for pdf Output


  • IDS will allow users to show/hide the property with their values in the description for pdf output.
  • By default, IDS shows property description in English form
  • By using “-hide_pdf_prop” none or “-hide_pdf_prop “<space separated property list>”” or -hide_pdf_prop all switch in the command line, user can hide property English description and shows the property name with their values in description.
  • Scope:-
    • This switch will support all input and it should be added in the command line.
    • For IDS GUI flavor like:- IDS-Excel/IDSWord/IDSNG, the user will have to add this switch in the command-line options box.
    • This feature supports only for pdf outputs.
    • For other output, the description will remain the same as previously.
  • Details Introduction:-
    • By using “-hide_pdf_prop” or “-hide_pdf_prop “<space separated property list>””  or -hide_pdf_prop all switch in the command line, user can hide property English description and shows the property name with their values in the description in pdf outputs. 
    • Usage of this switch:-
      • If the user wants to hide some specific property English description and shows all property names with their respective values then they can specify the property name with switch.

Like below:-

           <CMD>IDSbatch <testcase name > -out pdf -hide_pdf_prop “count,reset”


      • If the user wants to show all property English description with all property names along with their respective values then the user will have to add only switch in command line with arguments(Value)

Like below:-

                 <CMD>IDSbatch <testcase name > -out pdf -hide_prop_prop none


      • If the user wants to hide all property English description and shows all property names with their respective values then the user will have to add only switch in command line with arguments(Value)

Like below:-

                       <CMD>IDSbatch <testcase name > -out pdf -hide_pdf_prop all


Note:-  this switch by default hides the all property English description and shows the property name with their values.


RDL Input with command :



Command will be :- IDS-Batch <input file name> -out pdf -hide_pdf_prop all
