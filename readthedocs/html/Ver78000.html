RTL Enhancement


  1. B#2737: Support for APB aggregation logic for blocks with repeat VHDL output. More Details
  2. B#2738: Support for AXI aggregation logic for blocks with repeat VHDL output. More Details
  3. B#2739: Support for AMBA-AHB aggregation logic for blocks with repeat VHDL output. More Details
  4. B#2814: Support for aggr_block_offset in VHDL output. More Details
  5. B#2822: Support for aggregation logic only for the top module in case of Verilog output. More Details

General Enhancement


  1. F#23305: Support for removing the displayed time of generation in logs. More Details
  2. F#23269: Support for adding a formatted header including the company logo to every page header in the PDF document. More Details
  3. B#2823: Support for module name & dynamic assignments in Python output. More Details
  4. B#2824: Support for “top_display_name=true;” to flatten the instances in case of repetition for HTML and python output. More Details
  5. B#2827: Support for “profile.json”, for which each of the sw/hw access pair of combinations, number of fields existing will be shown in the file “profile.json” . More Details

Bug Fixes


RTL


  1. F#23413: Fix for IDS-Batch producing SystemRDL from IP-XACT which does not have expected addressing.
  2. F#23128: Fix for RAL generation crashing issue using IPXACT description file.
  3. B#2788: Fix for not checkout license issue in IDS-Word and IDS-Excel above ALM 2.7.1. 
  4. B#23385: Fix for improper syntax in the description, unopened or closed single curly brackets.
  5. F#23425: Fix for the clock_name property support with an alias register.
  6. F#23476:
    1. Fix for clock_name property with wide register
    2. Fix for clock_name property with wide register for low power switch 
  7. F#23216  - Fix for the lock call back issue when the lock register only has a signal in lock expression.
  8. F#23084 - Fix for the property “rtl.hw_set” combination with parity property when SW access = ro and  HW access = na.
  9. F#23248: Fix for lint issue for read-only registers with CRC
  10. B#2371: Fix for an error in case of “Invalid_addr_err=read” property.
  11. F#23432: Fix for lint error "The case statement with selector '{haddr[1:0],hsize[1:0]}' within the 'ahb3lite_widget' module lacks a default clause, rendering it incomplete".
  12. B#2816: Fix for clock gating enable condition, wire “_in_enb” is missing from the reg_in_enb equation.
  13. B#2820: Fix for the property “svheader_opt=1” not applicable on to the leaf level registers, user defined property “svheader_opt=1” is now supported with “-if_svheader” switch.
  14. B#2817: Fix for lint error in generated RTL, the "paddr<= 0" has been removed  which is causing the lint error using the top property "power_opt=4".

UVM


  1. B#2774: Fix for lock call back classes in uvm when lock property is applied on register with chip-in-chip case.


General

  1. B#2819: Fix for gcc issue in case of CHeaders, gcc throwing errors due to the compilation issue. 
  2. B#2585: Fix for pre-processing the RDL files to catch the issues upfront, warning inclusion instead of error.
