Customizing clock signal

default_clock_name

 

The rtl property default_clock_name is used to change the default name of the clock. The default name of the clock is clk and the user can write any string on its place like default_clock_name=clk_change.


Example:        IDS-NG        IDS-Word        IDS-Excel        SystemRDL


IDS-NG Register View



IDS-NG Spreadsheet View



SystemRDL


property clock_edge {type = string; component = addrmap|reg;  };

property default_clock_name {type =string; component = addrmap ; };

property chip {type=boolean; component = addrmap;};


addrmap chip_1 {

  chip = true;

  default_clock_name = "clk_change" ;

    addrmap block_1 {

      clock_edge = "negedge" ;

        reg reg_1 { 

         regwidth = 32;

          field {

           hw = rw;

           sw = rw;

           onread=r;

           onwrite=w;

          } F1[31:0] = 32'h0;

        };

        reg_1 reg_1 @0x0;

    };

    block_1  block_1 @0x0; 

};

clock_edge


The rtl property clock_edge is used to change the edge of the clock. The default edge of the clock is posedge and the user can give expicitly clock_edge=posedge/negedge. The example below shows that the property is used so the whole code will be changed with property string. 


Example:        IDS-NG        IDS-Word        IDS-Excel        SystemRDL


IDS-NG Register View



IDS-NG Spreadsheet View



SystemRDL


property clock_edge {type = string; component = addrmap|reg;  };  


// chip : chip1

addrmap chip1 {

  addrmap block1 {

    clock_edge = "negedge" ;    

    reg reg1 { 

       regwidth = 32;

      field {

        hw = rw;

        sw = rw;

      } F1[31:0] = 32'h0;

    }; 

    reg1 reg1 @0x0;

  };

  block1  block1 @0x0;

};

Removing buffer delay in clock assignment:


Property named ‘reassign_bus_clock’ with value ‘true’ can be used at the top component to overcome the buffer delay caused when an VHDL based widget is connected with a verilog RTL. This property  reassigns the proprietary clock signal with the bus clock signal in the verilog RTL  and also removes the connection of the proprietary clock ‘clk’ signal in rtl with the VHDL widget. 


Note: 

  1. This is a top property which can be applied only on either chip or block template.
  2.  The current  support is only for AXI buses.


Example: IDS-NG          IDS-Word          IDS-Excel          SystemRDL
