CSV

Comma Separated Values


Comma Separated Values (CSV) files can be generated by any spreadsheet tool (like Excel or OpenOffice Calc or Google Sheets) or even with a text editor or even automatically using scripts. Being plain ASCII text, these files provide a simple way for data communication. 


IDesignSpec™ can read and write CSV files in all its flavors : IDS-Word, IDS-Excel and IDS-Batch.


IDesignSpec™ supports a default CSV file format, however, in many cases, user can specify the format used in another (CSV) file. This meta information file (called a template file) allows more flexibility to the user.


Note that almost all functionality that is supported by IDesignSpec™ works through the CSV work flow as well.


Here is a sample...


chip,block,section name ,register,field,sw access,hw access,field default,bits

chip1,,,,,,,,

,bk1,,,,,,,

,,reggrp1,,,,,,

,,,reg_11,,,,,

,,,,fld_1,rw,na,,1

,,end section,,,,,,

,,,reg_31,,,,,

,,,,fld_1,ro,,0x03,5:3

,,,,fld_2,rw,,,1

To generate outputs, there are two possibilities. Either the CSV file can be opened in Excel where IDS-Excel plug-in has been installed, or, the CSV file can be directly processed using IDS-Batch. 


1. Using IDS-Excel


  • In order to generate outputs using IDS-Excel, a user customizable "IDS_template.xlsx" sheet is required. 
  • The IDS environment variable, “IDSEXCEL_TEMPLATE_FILE” pointing to the location of this "IDS_template" sheet should be defined. It enables the entire team to use the same format.
  • To create a custom template, start with a template provided inIDS Docs/examples/excel2007 bundle. Download the "Supporting Files" (IDSDocs) which contains sample templates and examples from Agnisys® download site.
  • Click on the 'Check' button to perform automatic checks. Note that if there are errors, appropriate messages will be displayed next to the source of errors. 
  • Create all desired outputs by clicking  on the 'Generate' button.



2. Using IDS-Batch


The usage syntax of IDS-Batch is shown below: 


IDSbatch [optional arguments] [–out “format(s)”] [-template_file "<path to "IDS_template" sheet>”] input_csv_file 


Run IDS-Batch command on command line for all supported options. For more details on IDS-Batch refer here.

Sample Runs:


  • Verilog and XRSL output using CSV input file

IDSbatch -out "verilog xrsl" -bus amba DesignSpec.csv


  • Using customized "IDS_template" sheet

IDSbatch -out xrsl -template_file D:\IDS_examples\IDS_template.csv DesignSpec.csv
