Custom CSV

(Compatible in IDS version 5.16.8 and above)


IDS supports import of a Custom CSV file.


IDS-Batch 


In IDS-Batch, user can generate the required outputs using a CSV format, through following command line-


IDSbatch <inputfile.csv> -out “<IDS outputs>” –csvheader “<custom header mapping>”


Note: 

  • "-csvheader" option is used to map user's CSV header with IDS header. This option is must with customCSV input file.
  • CSV header to be specified, should be present within double quotes. 


CSV Header format:


Format-

<IDS Header>:<user defined header>,<IDS Header>:<user defined header>………. N


Rules-


    • Input CSV file's header should be mapped to IDS Header(mentioned in table below) using colon(:).
    • Each header must be comma separated with no spacing.
    • Mandatory Headers are shown in bold in the table.
    • User Defined Header name should not start with a numeric value.
    • Name of IDS Headers mentioned in the table cannot be changed.
    • Any header in the input CSV file left unmapped, will convert to properties as “name=value” pair at the respective component level on import.
    • In CSV chip_name(if present) and block_name(if present) should be present repetitively to show the hierarchy. Repetitive hierarchy is optional for rest of components.   
    • If no block header is specified, all the data will be contained inside a block, which will have the same name as the input CSV file's name.

Default

IDSWord/IDSExcel


In IDS-Word/ IDS-Excel user can import the custom CSV file through "Import" window, as shown below. 

  • Select CSV from the “Import File type” option and specify the "CSV Header" (see below for CSV Header format)






CSV Header format:


To import CSV, user needs to mention CSV Header for mapping. It can be done using either of the 2 ways explained below-


1. Manually 


CSV Header can be specified by manual typing into the "CSV Header" column of "Import" window. User needs to follow a format and some rules which are explained above.


2. Through Configure Settings

Go to "Config-> CustomCSV" settings to map input CSV file's header with IDS as shown below-



  • Headers in CustomCSV file are placed in right textboxes.
  • For advance mapping make use of columns under the Properties section.
  • The value in Right textboxes are headers inside input CSV file
  • The value in Left textboxes will be the name for the corresponding input header in IDS template, after import completes.


Click 'OK' for CSV Header to reflect automatically in "IMPORT" window as shown below-
