Repeat on Memory

Introduction


A memory is a storage with contiguous address locations. When the size of memory is same and different interfaces with different ports are required then repeat property on memory template can be used.

generate //generate block for each repeat value

        genvar Mem1_i;

        for( Mem1_i = 0; Mem1_i < Mem1_count; Mem1_i = Mem1_i + 1)

        begin : Mem1_gen

    assign Mem1_decode[Mem1_i]         = (address[(addr_width - 1):7]  == Mem1_offset[Mem1_i][(addr_width -7) - 1 : 0]) ? 1'b1 : 1'b0;
