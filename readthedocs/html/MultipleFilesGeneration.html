Multiple Files Generation

Multiple RTL/UVM Output Files Generation

(Compatible with IDS version 5.14.0 and above for RTL outputs)

(Compatible with IDS version 5.14.24 and above for UVM output)


IDSWord/IDSExcel:


Despite generating a single RTL/UVM output, IDS now supports generation of multiple RTL/UVM files for a single IDS specification. Select the "Multi Out File" option in "Configuration Outputs Settings" to use this feature.



  • Verilog - Multiple Verilog Files(.v) is generated for each block and chip.
  • VHDL - Multiple VHDL files for each block are generated. Entity, package and Architecture for each block are present in separate files called "blockname_e.vhd", "blockname_pkg.vhd" and "blockname_arch.vhd" respectively.
  • UVM - Multiple SV files (.regmem.sv) for each block and chip alongwith a package file (topLevelModule_pkg.regmem.sv) are generated.
  • C header - Multiple Header files for each block and chip


IDSBatch:


In IDS-Batch the “-if” option will generate the individual files for RTL/UVM/Header output generation.


Example:  IDS-Batch  <file_name> -if –out “Verilog ”  -dir <output directory name>


Individual options for particular outputs


-if_header:  Generates seperate file for each block and chip [HEADER]

-if_uvm:  Generates seperate file for each block and chip [UVM]

-if_verilog:  Generates seperate file for each block and chip [VERILOG]
