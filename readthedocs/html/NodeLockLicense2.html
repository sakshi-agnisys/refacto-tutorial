Node Lock License

It is possible to get Node Lock License for your machine in two following ways.

1. Sending UUID to Agnisys® support

  • You must send the UUID for your machine to the support@agnisys.com, to get a node lock file. 
  • To get the UUID for your machine, click on 'About>Get UUID' button. A pop up will appear which displays the UUID for your machine.


Pop-up displaying the UUID for your machine



  • Once you have a license file for IDS-NG, you must set the environment variable 'AGNI_LICENSE_FILE' to point to this license file. You must restart IDS-NG if you have it running, to start using IDS. 
  • Now you are ready to open a new IDS-NG to put in specifications. 

2. Editor License

  • To get an IDesignSpec™ Editor license, download the latest IDesignSpec™ (Word/Excel) setup from the Agnisys® download page (login required).
  • After installation, click on the IDS ribbon’s "About" button, then click on the “Get UUID” button as shown in figure below:



  • Copy and send this UUID .txt to support@agnisys.com
  • Once a user has license file for IDesignSpec™, the environment variable 'AGNI_LICENSE_FILE' must be set to point to this license file. To start using IDS, MS Office must be restarted if it is currently in running state.
  • Users are now ready to open a new Word document to create specifications using IDS.
