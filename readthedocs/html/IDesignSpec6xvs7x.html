General Differences:


  1. “-rtl_wire”  switch is by default ON  in the 7.x version of IDesignSpec. 
  2. Changes in the code-ordering of the Generated RTL ( like some port’s declaration position changes in the port list). However the code is equivalent in 6x and 7.x versions. 
  3. Expressions in the Hdl_path must be enclosed in the “[  ]" in 7.x IDS version. 
  4. There are many features which are supported in the 7.x version only. The list of these features is available at:
    1. List of features supported in 7.x Verilog & UVM. 


Key Points: 


  1. RTL Generation time of 7.x is less as compared to 6.x version. 
  2. Both the IDesignSpec’s series i.e 7.x and 6.x are equivalent by logic, however there are some diffs due to change in code ordering.  
  3. IDesignSpec 7.x series  has more features as compared to the 6.x series.
