Removal of Write/Read ports Generated for Read-only/Write-only External CSRs


IDS will remove all writable ports for an External Csr in case of read only access and remove all readable ports in case of write only access. But in case of memory (type=mem) component IDS generate all the ports .

This will be done by  ‘rtl.strict_ports=true’(top property) for Verilog/SV Output.


Case1 : 

When both external csr are read only


Example: IDS-NG        IDS-Word        IDS-Excel        SystemRDL


IDS-NG Register View



IDS-NG Spreadsheet View




SystemRDL


propety rtl_strict_ports  {type = boolean ; comonent = addrmap;};

addrmap block1 {

  rtl_strict_ports = true;

  regfile {

   reg R1 {

    field {

           sw = r;

           hw = rw;

   }F1= 1'b0;


  };

  R1 R1;

  R1 R11;

  }external rg1;

  regfile {

  reg R2 {

    field {

       sw = r;

       hw = rw;

    }F1= 1'b0;


  };

  R2 R2;

  }external rg2;


};



Outputs:


RTL 


module block1_IDS(

    

    // SECTION(EXTERNAL) : RG1 PORT SIGNAL

    rg1_rd_ack_in,

    rg1_rd_data_in,

    rg1_rd_valid_out,

    

    // SECTION(EXTERNAL) : RG2 PORT SIGNAL

    rg2_rd_ack_in,

    rg2_rd_data_in,

    rg2_rd_valid_out,


    .   .   .   .    .


    assign ext_rd_data = (external_rd_ack == 1'b1)? ( rg1_rd_data | rg2_rd_data):     {bus_width{1'b0}} ;

    

    assign external_rd_valid = rg1_rd_valid | rg2_rd_valid; // END external_rd_valid

    

    assign external_rd_ack = rg1_rd_ack_wire | rg2_rd_ack_wire; // END external_rd_ack

    

    assign external_access = (rg1_decode_wire | rg2_decode_wire) & (rd_stb | wr_stb); // END external_access

      

    .   .   .   .    .


    assign request = 1'b1;

    assign rd_data_vld = external_rd_ack |(~external_access & rd_stb & rd_wait_state);

    assign rd_wait = rd_wait_state;

    

  .   .   .   .    .


endmodule

endmodule
