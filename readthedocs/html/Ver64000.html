Ver 6.40.0.0

v6.40.0.0 

(June 30th,2020)


IDesignSpec™ (IDS)


RTL Enhancements:


  • “wide_external_access=true” property has been supported for direct access to external memories having width greater than the bus width. (More Details)
  • I2C has been enhanced with target address and register address. Target address can be provided through input port instead of parameter.
  • "rtl.bmax" and "rtl.bmin" has been supported in VHDL also. Earlier, it was supported in verilog only (More Details)



UVM Enhancements:


  • Multiple reset values of fields have been supported in case of repeated register. (More Details)
  • "auto_mirroring" has been supported for third party RTL in both UVM RAL and ARV™ (More Details)


CHeader Enhancements:


  • "cheader.initialization" property has been supported for initialization of registers with default value which helps in reading default value of register in the reset condition (More Details)



SystemRDL Enhancements:


  • “reg_sw” and “reg_hw” UDP has been supported for SystemRDL input.


General Enhancements:


  • Support for adding depreciated message for feature list (HTML alt1 , ERM, Systemalt1, Wordalt1) in IDS-Batch CLI, IDS-Word, IDS-Excel, IDS-NG. Also, turbo has been removed from IDS-Word
  • Support for additional environment variable, AGNILM_LICENSE_FILE, in flex licensing to specify license server/file in IDS-Word.


Bug Fixes:


  • Fixes done for generation of fillers with unique name in case of regalign.
  • Fixes done in case of single field, for register struct generation with big endian and little endian.
  • Fixes done for bus width value in SystemRDL input in case of “refPath” UDP.
  • Fixes done for CSS vertical alignment issue in HTML and HTML alt2 outputs.
  • Flexlm server daemon is enhanced to support Red hat 6 onwards.
  • Fixes done in custom coverage in uvm output.
  • Fixes done for interrupt in case of ref in uvm output.
  • Fixes done in "-verical_reuse" command line switch along with "rdl_define" property.
  • Fixes done in "uvm.reg_access" property.
  • Fixes done for changing default endianness configuration in UVM ouput.
  • Fixes done for ref on registers and section with dynamically applied ispresent.
  • Fixes done in paged register properties, i.e., "page_wr_select" and "page_rd_select".
  • Fixes done for different bus_width for multiple blocks in chip.
  • Fixes done for removing redundant prefix from register name in a struct.
  • Fixes done in struct in case of repeat on the block.


IDS-NG


Enhancement:


  • Enhancement for adding quick documentation link in menu help, i.e, Quick Start (More Details)


Bug Fixes:


  • Fixes done for users define TCL file checkbox issue on project change. (More Details)
