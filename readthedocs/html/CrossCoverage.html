Cross Coverage

Cross Coverage 


A coverage group can specify cross coverage between two or more coverpoints or variables. Cross coverage is specified using the ‘cross’ construct. In IDS, we have a property called ‘cross’.


Note: IDS Property ‘cross’ is only supported with “coverage=on” and does not work with repeat.

Syntax:

  • On Register

{cross = <field_name> : <field_name> }

{cross_wr = <field_name> : <field_name> } for creating coverpoint inside wr_cg_vals

{cross_rd = <field_name> : <field_name> }  for creating coverpoint inside rd_cg_vals

  • On Block

{cross = <Register_name>.<field_name> : < Register_name>.<field_name> }

Cross coverage Bins Addition


In case of cross coverage the coverpoint can be adding using uvm custom code property [uvm loc=cross.<Coverpoint name>] <bins> [/uvm].The Coverpoint name in uvm loc should same as provided in IDS cross property.
