Python Data Dump

Python is a general-purpose high level programming language.  With this output it will generate a Python data structure in the form of dictionary in dictionary. It will have all the information about the design. User can use their python program to get the information from this data structure.


Example


IDSWord




IDSExcel

The table below gives the description of some keys which are internally generated by IDS.


Keys

Description

type

Type of the element

emptyspace

Empty space in the addressmap

sheetname

Name of sheet in excel / Name of top element in RDL

absolutepath

Absolute path of ref file in Ref case.

orig_path

It is path shown in the template.

xrsl_path

Absolute path of xrsl file in ref case.

link

Name of the element which is being ref.




For every property supported by IDS, a new key is formed in the data dump. To know these properties, refer to "Properties section".



Support for generating Python output in multiple files for large IPs


Users can now generate Python output in multiple files for large IPs by using the "-if" flag in the command line. When the user provides the "-if" flag in the command line and generates Python output, separate files will be generated for each chip and block.


Example: IDS-NG        IDS-Word        IDS-Excel        SystemRDL


Command Line


 idsbatch “test.rdl” -if -preserve -verbose -dir “ids” -out “python”

IDS-NG Spreadsheet View



SystemRDL


property chip {type=boolean; component=addrmap;};


addrmap LPDDR_reg{

  reg regA{

    field {}f1;

  };regA regA;

};


addrmap DDR_reg{

  reg regA{

    field {}f1;

  };regA regA;

};


addrmap top{

  chip=true;

  LPDDR_reg LPDDR_reg;

  DDR_reg DDR_reg;

};

Chip in chip support for Python output


Users now have the ability to generate Python output for a chip in a chip scenario.


Example: IDS-NG        SystemRDL


IDS-NG Register View




IDS-NG Spreadsheet View


               



               


SystemRDL

property chip {type=boolean; component=addrmap;};

addrmap chip2{

  chip=true;

  addrmap block1{

            reg reg1{

                field {

                   sw=rw;

                   hw=rw;

                }f1[31:0]=0;

            };reg1 reg1;

  };block1 block1;

};

addrmap chip1{

chip=true;

chip2 chip2;

};
