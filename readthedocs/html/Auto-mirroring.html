Auto-mirroring

Auto-mirroring 

Whenever a HW event occurs, the value on the HW interface needs to be updated on the Regmap. To support this, the feature of Auto-mirroring in IDS generated UVM RAL is introduced.

Auto-mirroring updates the UVM Register Field with the value on HW interface when a HW event occurs on it. The Required value is tapped from inside the HW Register interface through a Block HW interface.

The feature of auto-mirroring is enable using command line switch “-auto_mirror”


In the diagram below, model_updater will call a task hw_update_mirrors i.e. a collection of all register’s hw_update tasks. 

In each individual task a hardware event is being monitored and when HW event occurs it will update the value on that particular register in the register map.


Users have to create the model_updater class instance in their UVM environment and also have to set HW interface and RAL model in UVM environment.

P.S . Auto-mirroring is Currently supported Chip level.

Below are the files that are created while using “auto-mirror”:

  1. <RAL_MODEL>.regmem.sv
  2. Model_updater.svh
  3. Hw_if.svh
  4. HWif_access_block


Steps for integrating Auto_mirroring in testbench:


  • Run command :

 IDS-Batch <inp_file>  -out uvm -dir <out_dir>  -auto_mirror


  • Build model_updater class in UVM test

P.S. add the model_updater.svh file in test_pkg

  • Connect the register signals with block_hw interface in module HWif_access_block.


module HW_intf_access_block;

import uvm_pkg::*;


Block1_hw_if Block1_hw();


/*  Replace "User_RTL" on the right side with the hierarchical register port with HW_interface signal on left

example:  assign Block1_hw.Reg1_F1_in_enb        = wrapper.block1.Reg1_F1_LOAD;

assign Block1_hw.Reg1_F1_in        = wrapper.block1.Reg1_F1_IN;  */


//port connection for Reg1_F1


assign Block1_hw.Reg1_F1_in_enb   = User_RTL.Reg1_F1_in_enb;

assign Block1_hw.Reg1_F1_in       = User_RTL.Reg1_F1_in;


//port connection for Reg2_F2


assign Block1_hw.Reg2_F2_in_enb   = User_RTL.Reg2_F2_in_enb;

assign Block1_hw.Reg2_F2_in       = User_RTL.Reg2_F2_in;


initial

begin

    uvm_config_db #(virtual Block1_hw_if)::set(null, "*", "Block1_hif",Block1_hw);

end


endmodule


P.S. Replace the signals on right side of assign statement with hierarchical register signals of DUT.

i.e.,

Block_hw.Reg1_F1_in_enb : Hw enabling signal.

Block_hw.Reg1_F1_in          : Writing value of the register.

Sample code :


class model_updater extends uvm_component;

    `uvm_component_utils(model_updater)


    virtual Block1_hw_if tapper_hif;

    Block1_block rm;



    function new (string name, uvm_component parent = null);

        super.new(name, parent);

        if (!uvm_config_db #(Block1_block)::get(null, "uvm_test_top*", "ral_model", rm)) begin

            `uvm_info("FROM_MODEL_UPDATER","RAL MODEL NOT SET",UVM_LOW)

        end

        if (!uvm_config_db #(virtual Block1_hw_if)::get(null, "*", "tapper_hif", tapper_hif)) begin

            `uvm_info("FROM_MODEL_UPDATER","TAPPER INTERFACE NOT SET",UVM_LOW)

        end


       

    endfunction : new


    task hw_update_Reg1();

        uvm_reg curr_reg = rm.get_reg_by_name("Reg1");


        uvm_reg_field F1 = curr_reg.get_field_by_name("F1");


        fork

        begin

            forever@(posedge tapper_hif.Reg1_F1_in_enb)

            begin

                do

                begin

                    void'(F1.predict(tapper_hif.Reg1_F1_in));

                end

                while(tapper_hif.Reg1_F1_in_enb);

            end

        end

        join_none

    endtask


    task hw_update_Reg2();

        uvm_reg curr_reg = rm.get_reg_by_name("Reg2");


        uvm_reg_field F2 = curr_reg.get_field_by_name("F2");


        fork

        begin

            forever@(posedge tapper_hif.Reg2_F2_in_enb)

            begin

                do

                begin

                    void'(F2.predict(tapper_hif.Reg2_F2_in));

                end

                while(tapper_hif.Reg2_F2_in_enb);

            end

        end

        join_none

    endtask


    task hw_update_mirrors();

        hw_update_Reg1();

        hw_update_Reg2();

    endtask


    task run_phase(uvm_phase phase);

        hw_update_mirrors();

    endtask

endclass : model_updater
