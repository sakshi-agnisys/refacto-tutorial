Custom XML



IDesignSpec™ can import a custom XML file into a register specification in Word format. The custom XML file can also be read through IDS batch mode. With the help of a user-defined xquery file, the Custom XML format can be converted to the IDS XRSL format


IDSWord


Note-

  • An example of using an xquery file to import a Custom XML file can be found in IDS Supporting Files. This folder is available for download from the Agnisys® website.
  • Xquery file is expected to be written by user with respect to its Custom XML input file.

IDSBatch


The following command is used to process a CustomXML file using xquery file and generate outputs.


IDSbatch <CustomXML_file> -ixq <XQuery_file> -out "xrsl"


Now, use the XRSL generated in above command to further generate other outputs, example-


IDSbatch <generated_xrsl> -out "verilog vhdl html" -bus amba
