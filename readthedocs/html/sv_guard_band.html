sv_guard_band


This property will generate the guard band in block interface file “blockname_if.sv” and the “wrapper file” which “top sv file” . The problem of multiple declaration of signal wr_data_out & address_out (in external_if.sv and Blockname_if.sv) can be resolved by using this property. This property is applied on Block.  The possible value for this property is true/false, e.g., {sv_guard_band=true}
