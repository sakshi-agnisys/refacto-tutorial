Register Access

Note : In addition to the above, SW and HW access can be "na" that is ― NO ACCESS. This ensures that the SW or HW does not have any access to the register field.

Custom Field Accesses in UVM RAL


IDS also supports SW accesses R/WOr, R/WAnd, WOr, WAnd for bitwise OR and bitwise AND operation on the current field value and the write data.


  • R/WOr - R: Read, W: Bitwise OR of current field value and write data.
  • R/WAnd - R: Read, W: Bitwise AND of current field value and write data.
  • WOr - W: Bitwise OR of current field value and write data.
  • WAnd - W: Bitwise AND of current field value and write data.
