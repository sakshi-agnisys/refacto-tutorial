VHDL Header

(Compatible with IDS version 5.14.24 and above)


This is a VHDL package consisting of some constants and records related to register spec. In order to generate this VHDL package file called "filename_const.vhd" select the "VHD Header" output under the Header section in "Configuration Outputs Settings" as shown below-



Example

IDS-Word


IDS-Excel


IDS-Batch

To generate VHDL header file through IDS-Batch, output tag "vhdheader" is added to "-out" option


Example

idsbatch -out "vhdl verilog uvm vhdheader html" -bus amba Design1.xlsx

VHDL Header Output

(compatible with IDS version 5.16.28 and above)


Now, the VHDL header output is generated compliant to 2008 standard.


LIBRARY IEEE;

USE IEEE.STD_LOGIC_1164.ALL;


package VHDL_Block_header_pkg is


    constant VHDL_BLOCK_ADDR : std_logic_vector(1 downto 0) := 2x"0" ;

    constant VHDL_BLOCK_SIZE : std_logic_vector := x"4" ;


    constant VHDL_BLOCK_REG1_ADDR : std_logic_vector(1 downto 0) := 2x"0" ;

    constant VHDL_BLOCK_REG1_DEFAULT : std_logic_vector(31 downto 0) := 32x"0" ;


    constant VHDL_BLOCK_REG1_FLD1_MSB : integer := 31 ;

    constant VHDL_BLOCK_REG1_FLD1_LSB : integer := 16 ;

    constant VHDL_BLOCK_REG1_FLD1_DEFAULT : std_logic_vector(31 downto 16) := 16x"0000" ;


    constant VHDL_BLOCK_REG1_FLD2_MSB : integer := 15 ;

    constant VHDL_BLOCK_REG1_FLD2_LSB : integer := 0 ;

    constant VHDL_BLOCK_REG1_FLD2_DEFAULT : std_logic_vector(15 downto 0) := 16x"0000" ;


-------------------------------------------

--Reg: VHDL_BLOCK/REG1

-------------------------------------------


type VHDL_Block_Reg1 is record 

    Fld1 : std_logic_vector(31 downto 16);

    Fld2 : std_logic_vector(15 downto 0);

 end record;


end package;
