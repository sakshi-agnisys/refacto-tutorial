Known Limitations


Known Limitations in IDesignSpec™

1. For Verilog, VHDL and SystemC outputs the IDS document cannot contain registers of different Reg Width in a same block. However, it is possible to generate other outputs having different Reg Width. 

2. External RegGroup should be located at natural addressable boundaries, that is, the address of the external RegGroup should be a multiple of its size. This restriction enables Log2 (Size) address bits to be used for addressing the RegGroup. See ― "External RegGroup -> Alignment" for more details. 



Known Limitations in ISequenceSpec™

  1. Nested "for" loops have not implemented
  2. Seq Index is not implemented with matlab output
  3. Read command have not implemented(Implicit read is supported)
  4. Only Z-Board is supported. (Other boards are in to do list)
  5. Signals and structs are only supported in UVM and Firmware output
  6. use of the register having repeat is not supported for all the outputs
  7. use of structures in the sequence is not supported for MATLAB output


Refer to individual APP-notes for additional limitations, if any.
