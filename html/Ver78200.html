Bug Fixes


RTL


  1. B#2852: Fix for the lock feature along with the disable_overlap_check_field flag.
  2. B#2853: Fix for regwidth issue with overlapping fields in condition mode.
  3. B#2869: Fix for reg_in_enb wire not defined in the IDS generated RTL. 
  4. B#2818: Fix for non generation of widget file in case of Verilog output.
  5. B#2750: Fix for svheader defines not generated correctly, blocks were getting generated with the same address offset. 
  6. F#23489: Fix for the issue related to the '*_parity_error_wire' signal in RTL when ref registers and top block  are used along with the "parity" property and repeat property.
  7. F#23294: Fix for the issue signal *_error” when external_byte_acc_error property is applied to register with sw=wo hw=na case.
  8. B#2872: Fix for annotation addition for the illegal combination of property “external=true” with “rtl.hw_clear=true” property.
  9. B#2873: Fix for the issue of default value of “rtl.hw_set=true” property when sw=rc and hw=rw when “rtl.hw_set=true” property applied to the field of the register with “ rtl.hw_enb=true” applied on the same register along with “parity=odd/even” property on the block.
  10. B#2874: Fix for the issue of extra “end” generated in RTL when register accesses are  sw=rw hw=wo with parity property applied to the block.
  11. F#23395/F#23394: Fix for the issue in SystemVerilog  with file `include upon using chip inside chip with multiple instances of the same file.
  12. F#23509: Fix for an issue with the external_ack.v module file for `def and `undef.
  13. F#23084: Fix for the issue with the combination of “rtl.hw_enb=false” property with the parity property and combination of “rtl.hw_clear/set” property with parity property. 
  14. F#23570: Fix for the lint error in SystemVerilog code generated for parameter bus_width definition of block offset, and it is equivalent to Verilog. 
  15. F#23496: Fix for CRC property applied on register.
  16. F#23567: Fix for SV output when using multiple instances of the same component, the include file calls another verilog module but instantiates a different module which does not exist.
  17. F#23512: Fix for HTML output sidebar navigation not navigating to registers.
  18. F#23474: Fix for Japanese characters not getting displayed correctly in PDF output.
  19. F#23204: Fix Using HTML tags for documentation in description and custom properties.
  20. B#2896: Fix for separate generation of vhdlalt1 and vhdlalt2 output.
  21. B#2855: Fix  dir_out_specific for vhdl output


General


  1. B#2870: Fix for moving the error to warning for auto_volatile at various chip levels. 
  2. B#2871: Fix for idsbatch crash when path to the IP is very long. 
  3. B#2815: Fix for Cheader files not getting generated after idsbatch run, output directory was empty after successful idsbatch run.
