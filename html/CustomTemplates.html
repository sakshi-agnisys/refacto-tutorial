Custom Templates

It is possible to change the default templates that IDS comes with. 

USING WORD TEMPLATES


IDS-Word (2007, 2010 and 2013) enables users to change the formats of IDS templates (chip, block, registers, TOC etc.) that are inserted in the document. 

This app note shows how a user can change the template. This feature enables users to use a different template for any reason and share the template with others in a group or across the company.


Step 1: Get the current template


The default templates present in Supporting files can be downloaded from Agnisys® download site. Copy these files into a directory anywhere on the file system. Lets call this directory user_template_dir. Only those files that will be modified need to be in this directory, others will be picked up from their internal default location.


Step 2:  Modify the templates


The template files in the user_template_dir can be modified as required. Here are some guidelines for changing the format of the templates:

Step 3: Set environment variable


Create an environment variable called “IDS_TEMPLATE_PATH” and point it to the directory user_template_dir. Now restart Word.
