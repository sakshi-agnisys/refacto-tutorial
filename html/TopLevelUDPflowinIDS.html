Applied Component
- leaf chip : it is the parent chip hierarchy above a block level hierarchy
- top chip : it refer to the top chip hierarchy in the chip-inside-chip specification

"reset_type" is to be applied only on the leaf chip hierarchy

"external_ack" is to be applied only on the leaf chip hierarchy

"coverage" can be applied on top chip component or leaf chip component. This allow the top property value to be overridden at the leaf chip hierarchy.

"auto_volatile" can be applied only on the top chip component

"ipxact_compact" can be applied only on the top chip component

"doc_unused_remove" is to be applied only on the leaf chip hierarchy
