YAML


IDS support YAML input file for generating different outputs. Sample input file in YAML format is as follows:


node:

      type: chip

      name: chip_name

      doc: "This is chip doc"

      node:

      -

         type: block

         name: block_name

         offset: 

         doc: "this is block doc"

Inline Enum for Reg/Field in YAML


The YAML processes the enum by using the enum template at only the block level for input as well as output YAML and other outputs. Now, inline enums can also be used in addition to the previous functionality.
