Address Sorting

(Compatible with IDS version 5.16.0 and above)


It is now possible to write the register spec in IDS, with register placed in any order, irrespective of their offset/address. To achieve this functionality inIDS Word/IDSExcel, select the "Address Sorting" option available in "Configuration" Settings under "Formating" section. 


With IDS-Batch CLI, include "-addr_sort" option on command line.
