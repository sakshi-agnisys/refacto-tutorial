Parameter Name Format


'param.name_format' property has the capability to customize the name of parameter in the Verilog and SystemVerilog output. 


param.name_format :   %c%b%s%r


  • %c : Here 'c' is referring to CHIP name. It will add the chip name to the count parameter in the RTL and System Verilog. 
  • %b : Here 'b' is referring to BLOCK name. It will add the block name to the count parameter in the RTL and System Verilog.
  • %s : Here 's' is referring to SECTION/REGGROUP name. It will add the section/reggroup name to the count parameter in the RTL and        System Verilog. If there are many levels of section, then the user can choose which level of section name to be include in the name generation. The syntax is “%s1/%s2” here '1' , '2' is the level of the section.
  • %r : Here 'r' is referring to Register name. It will add the register name to the count parameter in the RTL and System Verilog.

addrmap Chip1 {

param_name_format = "%c_%b%_%s1_%s";

  addrmap Block1 {

    addrmap S1 {

      addrmap S2 {

        reg Reg1 { 

           regwidth = 32;

          field {

            hw = rw;

            sw = rw;

          } F1[31:0] = 32'h0;

        };

        Reg1 Reg1 @0x00;

      };

      S2 S2[3] @0x00;

    };

    S1  S1[2] @0x00;

  };

 

  Block1  Block1 @0x00;

};
