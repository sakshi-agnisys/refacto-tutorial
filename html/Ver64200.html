Ver 6.42.0.0

v6.42.0.0 

(July 31st,2020)


IDesignSpec™ (IDS)


RTL Enhancements:


  • "buffer_trig_reg" property is supported at field level and enhanced to remove the requirement for applying both read and write triggers on a register with 'rw' software access. (More Details)
  • In “rtl_hw_vector” property, which is used to create vectored output and input port of a complete register, input and registered output ports of registers will now be created as per the size of registers, regardless of the fields being reserved or used. (More Details)
  • AMBA3-AHBLITE bus has been supported in vhdl.



UVM Enhancements:


  • The SystemRDL property "hdl_path_slice" is supported with the property "hdl_path_internal". (More Details)
  • AMBA3-AHBLITE bus has been supported in vhdl for ARV™. (More Details)



SV Enhancements:


  • Property "sv_interface=struct:bus_struct" is enhanced to flatten out the widget ports. (More Details)
  • “sv_interface=struct” property is now supported for sections inside sections. (More Details)
  • Structs in SystemVerilog have been enhanced with support for signals. (More Details)
  • Enhancement of external_if.sv file in case of AXI Bus. Now the file name changes to "external_if_axi.sv"  for the axi bus so that there can be multiple externals for different busses.


HTML ALT2 Enhancements:


  • Formatting for “add_doc” and “add_doc_above” property is supported in HTML ALT2 and Word output as well. (More Details)



Bug Fixes:


  • Revival of "rtl_port_suffix" property in version 6.38.26.0, which was accidentally deprecated in 6.36.0.0.
  • Fixes in "addressing = regalign" and  "addressing=fullalign"  property in case of external registers inside the internal section. 
  • Fixes in naming convention registers when struct’s are used in SystemVerilog output. It no longer appends its parent hierarchy as a prefix.
  • Fix for external_ack module instance name with property “external_module”.
  • Fixes in aggregation logic in case of parameterized offset for AMBA-APB Bus.
  • Fix for hdl paths in case of subblock and repeated sections.
  • Fix for external register in ARV™.
  • Fix for AMBA3-AHBLITE bus port connection in ARV™.
  • Fix in generation of fillers in case of fixed instance size of referred instances in C-Header Alt1 output.
  • Fix for unregistered registers and pulses (read and write) when used along with “rtl_hw_vector” property in verilog.


IDS-NG


Enhancement:


  • Support import CSV and YAML file. (More Details)
  • Support Import/ Export Configuration in IDSNG. (More Details)
