External RegGroups


Training Video: Creating RTL Models

The External RegGroup has same properties as external register, and accepts repeat and size. The mapping of RegGroup to RTL is addr map. An addr map can be inside another addr map so Reg-group can be inside another RegGroup and can have multi-level hierarchy.The External RegGroup is not implemented by IDesignSpec™ and need to be implemented manually by setting the "external" property as "true" in the template. IDesignSpec™ provides appropriate handshake signals repeatedly, based on the number of registers that are repeated for the external RegGroup to be joined to the hardware logic generated by IDesignSpec™:


address_out,

raddress_out,

wr_data_out,

<RegGroup>_<reg>_wr_req_in, 

<RegGroup>_<reg>_rd_ack_in, 

<RegGroup>_<reg>_rd_data_in, 

<RegGroup>_<reg>_wr_valid_out, 

<RegGroup>_<reg>_rd_valid_out,
