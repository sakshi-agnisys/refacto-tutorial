Markdown Output

Markdown 

Markdown is a lightweight markup language for creating formatted text using a plain-text editor. 

IDS will allow users to generate markdown as output like other documentation output from the specification.


Example:


                 

Use Case

IDS Command line:

idsbatch <input_text_sample> -out markdown -dir output_dir_markdown


Or

idsbatch <input_text_sample> -out md -dir output_dir_md

IDS GUI 

For GUI, Currently, users will have to add -out “markdown” in the command line option section to the configuration window for IDS GUI flavour. I.e; IDS-Word, IDS-Excel, IDSNG. We will add this option in the Configuration window for all IDS GUI flavours.
