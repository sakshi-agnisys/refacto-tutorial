IDesignSpec™ (IDS)


RTL Enhancements


  1. F#22497  - Support for lock property with software access read-to-clear (rc/wrc) More Details
  2. F#22352  - Support for byte access errors for external registers when the "byte_enable" signal is not equal to 'hF specifically for register More Details
  3. B#2226 - Support of AMBA AXI5lite bus More Details
  4. B#2184 - Support of bus parity for AMBA5 More Details 


UVM Enhancements


  1. F#22620 - Support for “-reg_port_case” switch in uvm RAL. More Details


Bug Fixes


RTL


  1. F#22411 - Fixed intr_out property together with intr.per_channel for sv output
  2. B#2240  - Fixed the 1, 0 and false possible values for accumulator property.
  3. F#22551 - Fixed the annotation for two properties combinations, i.e., rtl_name_format="%r"` and `sharedextbus
  4. F#22552 -  Fixed the undeclared wires in verilog netlist when generating CRC verilog.
  5. F#22760 - Fixed the property rtl.sw_write=clear when applied on field.
  6. F#22650 - Fixed the parity property usage when repeat is applied using parameters along with it.

    

UVM


  1. F#22619 - Fixed the generated RAL for repeated parity register.
  1. F#22536 - Fixed the uvm compilation issue with 'incrthreshold'
  2. F#22721 - Fixed the compilation issue when the register is protected by a mask bit.

General


  1. F#16100 - Fixed the displayName in the generated output from input ipxact1_4 
  2. F#22417 - Fixed the displayName in input ipxact1_4 to ipxact2014 or ipxact1_5 output
  3. F#22618 - Fixed the issue of parameter for repeat in ipxact
  4. F#22424 - Fixed the single and multiple bit length and style for word-datasheet output


IDS-NG™

  1. F#22458 - Support for new -doc_hide_prop_exclude “p1,p2” option used together with -hide_htmlalt2_prop/-hide_pdfalt4_prop “all” option to hide all the properties used except the properties p1, p2. More Details
  1. B#2226 - Support addition of AMBA AXI5lite bus in configuration window. More Details

  3. F#21963 - Fixed the issue of ”SW access" properties not getting displayed correctly in Word Datasheet output by IDS-Excel
  4. F#22360 - Fixed the issue of generation encrypted widget file in case of Flex License.
  5. F#22402 - Fixed the CDC property issue in IDS-Excel.
