Ports for manipulating bits of fields

XORED

 

The rtl property “xored” makes changes in programs that Logical XOR of all bits in the field.

This property is used only for Verilog output supported with the switch “-fast”.

 

In the programs we have some signals that are created for the reg fields (depends upon the reg fields access). So by using this property on fields we are enhancing the use of reg fields in the code as per our requirement.


The example below shows that there are two registers in a block where the property “xored” is applied on reg1 and not on the other reg:


Example: IDS-NG        IDS-Word        IDS-Excel        SystemRDL


IDS-NG Register View



IDS-NG Spreadsheet View




SystemRDL


addrmap blockname {  

  reg reg1 { 

     regwidth = 32;

    field {

     xored = true ;

      hw = rw;

      sw = rw;

    } F1[31:0] = 32'h0;

  };

  reg reg2 { 

     regwidth = 32;

    field {

      hw = rw;

      sw = rw;

    } F1[31:0] = 32'h0;

  };

  reg1 reg1 @0x0;

  reg2 reg2 @0x4;

};
