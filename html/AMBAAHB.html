AMBA-AHB

IDS supports AMBA bus both in Verilog and VHDL. All the transactions to the registers pass through this AMBA widget. 



Name

Width

Driver

Function

hclk

1

Clock Source

This clock times all bus transfers. All signal timings are related to the rising edge of HCLK. 

hresetn

1

Reset Controller

This signal is normally connected 

directly to the system bus reset signal.

haddr

Configurable address width

(Default is 32)

initiator

System Address bus.

hwrite

1

initiator

When HIGH this signal indicates a write transfer 

and when LOW a read transfer.

hwdata

Configurable bus width

(Default is 32)

initiator 

The write data bus is used to transfer data from 

the initiator to the bus target during write 

operations.

hsel

1

initiator

target select Decoder Each AHB target has its own target select signal and this signal indicates that the current transfer is intended for the selected target.

htrans

2

initiator

Transfer type. Supported commands are

       Value    Command

       00           IDLE 

      01           BUSY

         10          NONSEQ

    11          SEQ

      

hburst

3

initiator

Transfer is a single transfer or forms part of a burst

hprot

4

initiator

Protection control signals

hready

1

target

When HIGH the HREADY signal indicates that a transfer has finished on the bus. This signal may be driven LOW to extend a transfer.


Note: target on the bus require HREADY as both an input and an output signal

hrdata

Configurable bus width

(Default is 32) 

target

The read data bus is used to transfer data from bus target to the bus initiator during read operations.

hsize

3

initiator

Byte enable signal used to indicate the size of the transfer

hresp

2

target

The transfer response provides additional information on the status of transfer.
