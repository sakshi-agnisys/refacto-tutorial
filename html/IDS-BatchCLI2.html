IDS-Batch CLI

On Windows


  • Download IDS-Batch from Agnisys® website (www.agnisys.com). Extract the downloaded file in a directory.
  • Add the location of extracted directory to the "PATH" environment variable (for example- D:\Agnisys\IDS Releases\IDSbatch).
  • Run "IDSbatch" from the command line.
  • Send the output of IDS-Batch –lic to Agnisys® customer care (support@agnisys.com) to get your license file (only for node lock license). 
  • Set AGNI_LICENSE_FILE variable to point to the license file or set the AGNI_LICENSE_SERVER variable to point to the license server (for floating license).
  • See Licensing section for more details.


On Linux and Mac


  • Download IDS-Batch from Agnisys® website (http://www.agnisys.com). Uncompress the download file in a directory.
  • Add the directory to your PATH variable by adding the following lines:
  • In .bashrc for bash shell 
  • % export PATH=<complete path of directory containing IDS-Batch>:$PATH
  • In .cshrc for cshell 
  • % setenv PATH <complete path of directory containing IDS-Batch>:$PATH 
  • Run IDS-Batch from terminal.
  • Send the output of IDS-Batch –lic to Agnisys® customer care (support@agnisys.com) to get your license file (only for node lock license). The output will be similar to the one shown in figure below:
