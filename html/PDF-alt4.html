PDF-alt4

Introduction:

    • IDS will allow users to generate pdfalt4 output .
    • PDF output is already there but generation for pdf output through XSLT generator is slow and has many limitations.


         Scope:-

    • It can be generated by using pdfalt4 output type
    • For IDS GUI flavor like- IDS-Excel/IDSWord/IDSNG, the user can generate by pdfalt4 by selecting an output from the Documentation.
    • This output will allow the user to generate multiple output files for chip/board level test case.
    • PDFAlt 4 output will allow the user to customize their output as they want. Change font size, color, font style for any attribute like Name, Offset, Size property. And all the formatting of SystemRdl and htmllat2 is supported in the pdfalt4 output.

Support for chip-in-chip


For generating PDF output for chip-in-chip case use pdfalt4 in -out switch. PDFalt4 and HTMLalt2 are exact copies of each other.


SystemRDL Input

property chip {type=boolean; component=addrmap;};

reg regA {

     field {}F1;

            };

addrmap block_1 {

        regA regA;

};

addrmap block_2 {

       regA regA;

};

addrmap block_3 {

       regA regA;

};

addrmap chip {

     chip=true;

   block_1 b1; 

   block_2 b2;

   block_3 b3;

};

addrmap top {

     chip=true;

     chip c1;

         };
