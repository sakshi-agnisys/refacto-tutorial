Memory Technology Mapping

The instantiation of memory modules of different vendors is possible in IDS by using the properties below.


Memory integration is supported for different vendors namely Synopsys, Altera, Xilinx.

NOTE: in this version we are supporting only for synopsys

 

IDS supports the instantiation of both types of memories - single and dual port memories.


 1. Single port memory: when either the sw_access or hw_access is “na”.

 2. Dual port memory: when the both the sw_access and hw_access is not “na”.

Memory Technology Mapping in IDS


How to enable this special feature in IDesignSpec™


IDS-Word/IDS-Excel


From the Advanced Design in the IDS configuration settings, select the option "Memory technology mapping"



IDS-Batch CLI


In the IDS-Batch CLI memory technology mapping option is enabled by specifying the "-memtechmapping" option in the IDS-Batch command line.
