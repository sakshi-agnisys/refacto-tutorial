RTL Enhancements


  1. F#22165 -  Support for parity error dependency on read and write valid “parity_error=<signal_name>:rw”. More Details
  1. F#22187- Support for parameterized values in lock expression of lock register. More Details
  1. F#22099 - Support for a special accumulator register. More Details
